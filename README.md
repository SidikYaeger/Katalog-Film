# Katalog Film - Tugas Besar Struktur Data 🎬

Repositori ini berisi kode sumber untuk aplikasi **Katalog Film**, sebuah program berbasis *Command Line Interface (CLI)* yang ditulis murni menggunakan bahasa **C++**. Proyek ini dikembangkan sebagai bentuk implementasi langsung untuk Tugas Besar mata kuliah Struktur Data.

## 🛠️ Teknologi yang Digunakan
- **Bahasa Pemrograman:** C++
- **IDE:** Code::Blocks

## 📌 Deskripsi Program
Program ini menggunakan konsep struktur data *(seperti Linked List / Tree / Graph — silakan sesuaikan)* untuk mengelola, menyimpan, dan memanipulasi informasi terkait film. Kode dibagi menjadi beberapa modul (`main.cpp`, `tubes.cpp`, dan `tubes.h`) untuk menjaga struktur kode tetap rapi dan mudah dibaca (Modular Programming).

## ✨ Fitur Utama
*(Catatan: Silakan sesuaikan atau hapus bagian yang tidak ada di program aslimu)*
- **Insert Data:** Menambahkan data film baru beserta detailnya.
- **Show/View Data:** Menampilkan seluruh katalog film yang sudah terdaftar.
- **Search:** Mencari film spesifik berdasarkan kriteria tertentu (misal: judul atau genre).
- **Delete:** Menghapus data film dari struktur memori.
